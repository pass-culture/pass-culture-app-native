 getBookingLabels
- should return the correct dateLabel for permanent bookings
- should not return the location for digital bookings
- should return the correct date and location for events

- should return the correct withdrawal date and location for events if starting tomorrow
- should return the correct withdrawal date if expires today for physical bookings
- should return the correct withdrawal date if expires tomorrow for physical bookings
- should return the correct dateLabel for digital bookings with activation codes but no expiration date
- should return the correct dateLabel for digital bookings with activation codes with expiration date
