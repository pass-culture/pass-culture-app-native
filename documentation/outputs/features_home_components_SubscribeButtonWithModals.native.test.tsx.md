 SubscribeButtonWithModals
- should open logged out modal when user is not logged in
- should show inactive SubscribeButton when user is logged in and not subscribed yet
- should show active SubscribeButton when user is logged in and already subscribed
- should show notifications settings modal when user has no notifications activated and click on subscribe button
- should show unsubscribe modal when user is already subscribed and click on subscribe button
- should show subscription success modal when user subscribe to a thematic for the second time
- should show snackbar when user subscribe to a thematic home for more than 3 times
- should not show anything when user is not eligible
