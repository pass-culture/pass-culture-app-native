useKeyboardEvents
 on iOS
- calls onBeforeShow props when calling keyboardWillShow
- calls onBeforeHide props when calling keyboardWillHide
- does nothing when calling keyboardDidShow
- does nothing when calling keyboardDidHide


 on Android
- calls onBeforeShow props when calling keyboardWillShow
- calls onBeforeHide props when calling keyboardWillHide


 useKeyboardEvents
- creates event listeners
- changes the event listeners when the props change
