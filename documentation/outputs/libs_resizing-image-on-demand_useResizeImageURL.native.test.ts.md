useResizeImageURL
 useResizeImageURL hook
- should return a smaller resized image URL on a small screen
- should return a larger resized image URL on a big screen
- should return a bigger resized image URL when the pixel density is 2
- should return the given image URL when the feature flag is off
- should return the resized image URL with custom dimensions when provided and height > width
- should return the resized image URL with custom dimensions when provided and width > height
- should return a bigger resized image URL with custom dimensions when provided and the pixel density is 2
