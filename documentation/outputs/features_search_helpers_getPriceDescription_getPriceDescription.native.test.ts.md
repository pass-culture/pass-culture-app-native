 getPriceDescription
- should return an empty string when minimum price is 0 and maximum price undefined
- should return "Gratuit" when minimum price undefined and maximum price is 0
- should return "Gratuit" when minimum price is 0 and maximum price is 0
- should return "de 5 € à 20 €" when minimum price is 5 and maximum price is 20
- should return "plus de 5 €" when minimum price is 5 and maximum price is undefined
- should return "moins de 5 €" when minimum price is undefined and maximum price is 5
