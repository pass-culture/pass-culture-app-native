TicketSwiper
 Swiper ticket controls
- should not show if number of ticket is equal to one
- should show if number of ticket is greater than one


 <TicketSwiper/>
- should display ticket without external bookings information if there are no external bookings (externalBookings is null)
- should display ticket without external bookings information if there are no external bookings (empty externalBookings array)
- should display one ticket with external bookings information if there are one external booking
- should display as many tickets as the number of tickets
