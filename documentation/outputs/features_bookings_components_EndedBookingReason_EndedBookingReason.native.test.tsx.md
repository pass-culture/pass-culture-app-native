 EndedBookingReason
- should return "Réservation utilisée" reason when date used defined
- should return "Annulée" reason when cancellation reason is OFFERER
- should return "Réservation archivée" reason when booking is eligible for archieve and cancellation reason is null
- should return "Réservation annulée" reason when booking was cancelled by beneficiary
- should return "Réservation annulée" reason when booking was expired
