AppButton
 * Icon property
- should display icon when provided
- should display icon when provided and icon position is left
- should display icon when provided and icon position is right
- should not display icon when not provided


 * isLoading property
- should display right elements when isLoading equals true
- should display right elements when isLoading equals false


 * Disabled property
- should disable handlers when disabled equals true
