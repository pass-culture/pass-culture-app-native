# EmailSent Component

## 🎯 Objectif
Le composant `<EmailSent />` affiche une interface utilisateur après qu'un e-mail a été envoyé, offrant la possibilité de consulter les e-mails.

## ⚙️ Comportements

### 1. Affichage initial
*   **Comportement:** Le composant affiche le bouton "Consulter mes e-mails".

### 2. Disponibilité de l'application de messagerie
*   **Comportement:** Le composant ne doit pas afficher le bouton pour ouvrir une application de messagerie si aucune application de messagerie n'est disponible sur l'appareil de l'utilisateur. (Ce comportement implique que l'affichage du bouton est conditionnel à la détection de la présence d'une application de messagerie).
