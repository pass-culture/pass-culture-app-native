# VenueDetails

Ce composant affiche les détails d'un lieu, en particulier la ville et le code postal, dans un formatage spécifique.

## Comportements et Scénarios

Le composant `VenueDetails` a le comportement suivant :

*   **Affichage des détails de lieu formatés** : Le composant doit afficher la ville et le code postal du lieu.

    *   **Règle de gestion:** La ville et le code postal sont combinés et formatés pour être affichés à l'utilisateur. L'implémentation exacte du formatage (par exemple, l'ordre, l'ajout de virgules ou d'espaces) n'est pas explicitée dans la description, mais le rendu doit être visuellement correct.
