# ThematicHighlightModule

Cette documentation décrit le fonctionnement du `ThematicHighlightModule`.

## Comportements

Le module gère un seul comportement.

*   **Interaction utilisateur**

    *   **Scénario:** Un utilisateur clique sur un `HighlightBlock`.
    *   **Comportement:** Le module doit consigner un événement `HighlightBlockClicked`. Cet événement peut être utilisé pour suivre l'interaction de l'utilisateur avec les éléments mis en évidence.
