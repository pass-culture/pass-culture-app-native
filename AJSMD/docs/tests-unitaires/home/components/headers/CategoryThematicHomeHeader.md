# CategoryThematicHomeHeader

## Description Générale

CategoryThematicHomeHeader est un composant d'en-tête thématique pour la page d'accueil des catégories. Il affiche un titre et un sous-titre.

## Comportements et Scénarios

### 1. Affichage Standard

*   **Scénario:** Le composant est rendu pour la page d'accueil des catégories.
*   **Comportement:**
    *   Le composant affiche correctement le titre et le sous-titre configurés.
