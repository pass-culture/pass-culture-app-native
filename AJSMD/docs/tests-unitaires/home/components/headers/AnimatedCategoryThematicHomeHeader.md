# AnimatedCategoryThematicHomeHeader

Cette documentation décrit le fonctionnement de la fonctionnalité `AnimatedCategoryThematicHomeHeader`. Il s'agit d'un composant d'en-tête animé pour la page d'accueil thématique d'une catégorie.

## Comportements et Scénarios

La fonctionnalité `AnimatedCategoryThematicHomeHeader` présente un comportement unique, basé sur la présence de titre et de sous-titre.

### Scénario : Affichage par défaut

*   **Objectif:** S'assurer que le composant affiche correctement les informations.
*   **Comportement attendu:**
    *   Le composant doit afficher à la fois le titre et le sous-titre.
    *   L'affichage est rendu correctement (sans erreurs de rendu).
