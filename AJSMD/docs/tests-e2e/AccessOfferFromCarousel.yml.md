# Exploration de Playlist Hybride

Ce document décrit les étapes suivies par un utilisateur pour explorer les fonctionnalités d'une playlist hybride.

## 1. Accéder aux Options d'une Playlist

*   L'application affiche une liste de playlists.
*   L'utilisateur appuie sur la playlist nommée "Playlist hybride 3".
*   L'utilisateur effectue un glissement vers la gauche sur cette playlist.
*   L'utilisateur appuie ensuite sur le bouton "En voir plus".

## 2. Vérification et Retour

*   L'application affiche l'écran de recherche.
*   L'utilisateur appuie sur le bouton "Revenir en arrière".
