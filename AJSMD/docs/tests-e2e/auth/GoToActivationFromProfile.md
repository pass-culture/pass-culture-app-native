# Création d'un compte et vérification du numéro de téléphone

Ce document décrit les étapes pour créer un compte dans l'application et vérifier votre numéro de téléphone.

## 1. Bienvenue et découverte de l'offre

*   L'utilisateur lance l'application et l'écran de bienvenue s'affiche avec le message "Bienvenue_!".
*   L'utilisateur appuie sur le bouton "Débloque tes 300_€".

## 2. Introduction et étape suivante

*   L'écran suivant affiche le message "C'est très rapide_!".
*   L'utilisateur appuie sur le bouton "Numéro de téléphone".

## 3. Conseils et informations

*   L'écran affiche des "Quelques conseils".
*   L'utilisateur appuie sur "J’ai compris".

## 4. Saisie du numéro de téléphone

*   L'écran demande "Quel est ton numéro de téléphone_?".
*   L'utilisateur appuie sur le champ "Entrée pour le numéro de téléphone".
*   L'utilisateur entre le préfixe "07".
*   L'application génère aléatoirement et affiche les 8 chiffres restants du numéro de téléphone.
*   L'utilisateur appuie sur le bouton "Continuer".

## 5. Vérification du numéro de téléphone

*   L'écran affiche le message "Valide ton numéro de téléphone".
*   L'utilisateur appuie sur le champ "Entrée pour le code reçu par sms".
*   L'application génère aléatoirement et affiche les 6 chiffres du code SMS.
*   L'utilisateur appuie sur le bouton "Continuer".

## 6. Profil et fin du processus

*   L'écran affiche le message "C'est très rapide_!".
*   L'utilisateur appuie sur "Profil non complété".
