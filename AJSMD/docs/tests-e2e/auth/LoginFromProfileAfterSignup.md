# Se connecter à l'application

Ce document décrit le processus de connexion d'un utilisateur non enregistré à l'application.

## 1. Accès à la page de connexion

*   L'utilisateur arrive sur l'écran principal de l'application.
*   L'utilisateur appuie sur le bouton "Mon profil".
*   L'utilisateur sélectionne l'option "Se connecter".

## 2. Saisie des identifiants

*   L'application affiche l'écran de connexion.
*   L'utilisateur entre son adresse email dans le champ prévu à cet effet.
*   L'utilisateur appuie sur le champ "Ton mot de passe".
*   L'utilisateur entre son mot de passe.
*   L'utilisateur ferme le clavier virtuel.
*   L'utilisateur appuie sur le bouton "Se connecter".

## 3. Gestion de la première utilisation

*   Si l'application affiche un écran "Prends 1 minute", proposant une option pour une prochaine utilisation, l'utilisateur appuie sur "Plus tard".
