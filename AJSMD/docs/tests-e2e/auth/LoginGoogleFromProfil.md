# S'inscrire via Google

Ce document décrit le processus d'inscription d'un utilisateur sur l'application en utilisant son compte Google.

## 1. Démarrage et Accès à l'Inscription

*   L'utilisateur vérifie qu'il voit son profil.
*   L'utilisateur appuie sur le bouton "Se connecter".

## 2. Connexion avec Google et Sélection du Compte

*   L'application affiche l'écran de "Connexion".
*   L'utilisateur appuie sur le bouton "Se connecter avec Google".
*   L'utilisateur sélectionne son compte Google.

## 3. Finalisation de l'Inscription

*   Si l'application affiche l'écran "Termine ton inscription", l'utilisateur procède aux étapes suivantes :
    *   L'utilisateur fait glisser son doigt sur l'écran pour sélectionner le mois précédent.
    *   L'utilisateur appuie sur "Continuer vers l’étape CGU & Données".
    *   L'utilisateur appuie sur "J’ai lu et j’accepte les conditions générales d’utilisation".
    *   L'utilisateur appuie sur "J’ai lu la charte des données personnelles".
    *   L'utilisateur appuie sur "Accepter les conditions générales d’utilisation et la politique de confidentialité pour s’inscrire".
    *   L'utilisateur choisit de "Vérifier mon identité plus tard".
