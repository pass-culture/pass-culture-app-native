# Connexion à l'application

Ce document explique les étapes de connexion à l'application pour un utilisateur existant.

## 1. Démarrage de la connexion

*   L'utilisateur appuie sur le bouton "Se connecter".
*   Si l'écran d'inscription s'affiche (ce qui peut arriver par erreur), l'utilisateur revient en arrière.

## 2. Saisie de l'adresse e-mail

*   L'utilisateur interagit avec le champ de saisie prévu pour l'adresse e-mail.
*   L'application affiche ensuite l'écran de saisie de l'email (le détail de ce processus est géré par une autre étape, décrite dans le fichier InputEmail.yml).

## 3. Saisie du mot de passe

*   L'utilisateur appuie sur le champ "Ton mot de passe".
*   L'utilisateur entre son mot de passe.
*   Le clavier est ensuite masqué.

## 4. Validation de la connexion

*   L'utilisateur appuie sur le bouton "Se connecter".
*   L'application attend la fin de l'animation de chargement.

## 5. Gestion des propositions additionnelles

*   **Si** une fenêtre proposant de répondre à un questionnaire de satisfaction apparaît ("Prends 1 minute"), l'utilisateur choisit de répondre "Plus tard".
*   **Si** une fenêtre proposant de partager l'application apparaît ("Partager l’appli"), l'utilisateur choisit "Non merci".
