# Essai de Connexion avec Échec

Ce document décrit le parcours d'un utilisateur qui tente de se connecter à l'application, mais rencontre une erreur.

## 1. Lancement de la Connexion

*   L'utilisateur appuie sur le bouton "Se connecter".

## 2. Saisie des Identifiants

*   L'application affiche l'écran de connexion avec le message "Connecte-toi".
*   L'utilisateur appuie sur le bouton "Afficher le mot de passe" pour visualiser ce qu'il tape.
*   L'utilisateur appuie sur le champ "Entrée pour l’email".
*   L'application génère aléatoirement une adresse e-mail et l'insère.
*   L'utilisateur appuie sur le champ "Ton mot de passe".
*   L'utilisateur tape le mot de passe fourni.
*   L'utilisateur ferme le clavier virtuel.

## 3. Tentative de Connexion et Échec

*   L'utilisateur appuie sur le bouton "Se connecter".
*   L'application affiche le message "E-mail ou mot de passe incorrect".
*   L'utilisateur appuie sur le bouton "Revenir en arrière" pour effacer les champs et recommencer.
