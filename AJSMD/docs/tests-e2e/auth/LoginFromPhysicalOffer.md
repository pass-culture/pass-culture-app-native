# Découverte et Réservation d'une Offre Culturelle

Ce document explique comment un utilisateur explore les offres culturelles disponibles dans l'application et tente de les réserver.

## 1. Affichage des Offres

*   L'application s'assure que les animations sont terminées pour afficher l'écran principal.
*   L'application vérifie que la section contenant les "tags" ou catégories est affichée.

## 2. Navigation dans les Catégories et Choix d'une Offre

*   L'utilisateur fait glisser son doigt vers le haut pour voir plus de catégories.
*   **Pour les utilisateurs Android :** L'application recherche et affiche le bouton "Envoyer par SMS".
*   **Pour les utilisateurs iOS :** L'application recherche et affiche le bouton "Envoyer sur iMessage".
*   L'application vérifie la présence d'un bouton "Voir l’itinéraire".
*   L'utilisateur appuie sur le bouton "Réserver l’offre".

## 3. Gestion de la Modalité de Connexion

*   L'écran affiche une fenêtre contextuelle ("modale") demandant à l'utilisateur de s'identifier pour bénéficier des offres.
*   L'utilisateur appuie sur le bouton "Fermer la modale".

## 4. Retour à l'écran principal

*   L'application s'assure à nouveau que les animations sont terminées.
*   L'utilisateur appuie sur l'icône de menu située en haut de l'écran, en général la deuxième icône de la barre de menu.
