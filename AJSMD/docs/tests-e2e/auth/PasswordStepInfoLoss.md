# Création d'un compte utilisateur

Ce document explique les étapes qu'un utilisateur suit pour créer un nouveau compte dans l'application.

## 1. Démarrage et accès à la création de compte

*   L'utilisateur arrive sur l'écran principal de l'application.
*   L'application affiche l'option "Mon profil".
*   L'utilisateur appuie sur "Créer un compte".

## 2. Saisie des informations personnelles

*   L'application affiche l'écran "Crée-toi un compte".
*   L'utilisateur appuie sur le champ "Entrée pour l’email".
*   L'utilisateur entre son adresse email non enregistrée dans l'application.
*   L'application masque le clavier virtuel.
*   L'utilisateur coche la case "J’accepte de recevoir les newsletters, bons plans et les recommandations personnalisées du pass Culture.".
*   L'utilisateur appuie sur "Continuer vers l’étape Mot de passe".

## 3. Création du mot de passe et abandon de la création de compte

*   L'application affiche l'écran "Choisis un mot de passe".
*   L'utilisateur appuie sur le champ "Ton mot de passe".
*   L'utilisateur entre un mot de passe.
*   L'application masque le clavier virtuel.
*   L'utilisateur appuie sur "Quitter".
*   L'utilisateur appuie sur "Abandonner l’inscription".
