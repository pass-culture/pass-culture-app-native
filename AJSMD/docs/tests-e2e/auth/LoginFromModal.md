# Se connecter à l'application

Ce document décrit les étapes pour qu'un utilisateur se connecte à l'application.

## 1. Lancement de la connexion

*   L'utilisateur appuie sur le bouton "Se connecter".
*   L'application affiche un écran avec le texte "Connecte-toi".

## 2. Saisie des identifiants

*   L'utilisateur appuie sur la zone de saisie "Entrée pour l’email".
*   L'application affiche une autre interface pour la saisie de l'email.
*   L'utilisateur saisit son email.
*   L'utilisateur appuie sur le champ "Ton mot de passe".
*   L'utilisateur saisit son mot de passe.
*   L'application ferme le clavier.

## 3. Connexion et vérification

*   L'utilisateur appuie sur le bouton "Se connecter".
*   *Si un écran de sondage apparait, l'utilisateur appuie sur "Plus tard".*
