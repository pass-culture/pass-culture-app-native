# Validation du Numéro de Téléphone

Ce document décrit le processus de validation du numéro de téléphone pour un nouvel utilisateur.

## 1. Démarrage et Informations Préliminaires

*   L'application affiche un message : "C'est très rapide !".
*   L'utilisateur appuie sur le bouton "Numéro de téléphone non complété".
*   L'application affiche quelques conseils.
*   L'utilisateur appuie sur "J’ai compris".

## 2. Saisie du Numéro de Téléphone

*   L'application affiche une question : "Quel est ton numéro de téléphone ?".
*   L'utilisateur appuie sur le champ pour entrer son numéro.
*   L'utilisateur entre l'indicatif téléphonique "07".
*   L'application génère automatiquement les 8 chiffres restants du numéro de téléphone.
*   L'utilisateur appuie sur le bouton "Continuer".

## 3. Validation du Numéro de Téléphone par SMS

*   L'application affiche un message : "Valide ton numéro de téléphone".
*   L'utilisateur appuie sur le champ prévu pour entrer le code de vérification reçu par SMS.
*   L'application génère automatiquement un code à 6 chiffres.
*   L'utilisateur appuie sur le bouton "Continuer".
