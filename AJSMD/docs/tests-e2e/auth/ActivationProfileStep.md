# Vérification de la rapidité d'une action et navigation vers le profil incomplet

## Introduction

Ce document décrit les étapes suivies pour vérifier la rapidité d'une action dans l'application et naviguer vers le profil utilisateur.

## 1. Vérification de la rapidité

*   L'application affiche le message "C'est très rapide !".

## 2. Accès au profil incomplet

*   L'utilisateur appuie sur l'élément "Profil non complété".
