# Création d'un compte utilisateur

Ce document décrit le processus de création d'un nouveau compte utilisateur dans l'application.

## 1. Démarrage et accès à l'inscription

*   L'utilisateur commence sur l'écran principal de l'application.
*   L'utilisateur appuie sur le bouton "Mon profil".
*   L'utilisateur appuie sur le bouton "Créer un compte".

## 2. Informations personnelles et acceptation des conditions

*   L'écran "Crée-toi un compte" s'affiche.
*   L'utilisateur appuie sur le champ "Entrée pour l’email".
*   L'utilisateur saisit son adresse e-mail.
*   L'application masque le clavier virtuel.
*   L'utilisateur coche la case pour accepter de recevoir les newsletters et recommandations.
*   L'utilisateur appuie sur le bouton "Continuer vers l’étape Mot de passe".

## 3. Définition du mot de passe

*   L'écran "Choisis un mot de passe" s'affiche.
*   L'utilisateur appuie sur le champ "Ton mot de passe".
*   L'utilisateur saisit le mot de passe souhaité.
*   L'application masque le clavier virtuel.
*   L'utilisateur appuie sur le bouton "Continuer vers l’étape Date de naissance".

## 4. Saisie de la date de naissance

*   L'écran "Renseigne ta date de naissance" s'affiche.
*   L'utilisateur fait glisser l'écran pour sélectionner le mois de naissance.
*   L'utilisateur fait glisser l'écran pour sélectionner l'année de naissance.
*   L'utilisateur appuie sur le bouton "Quitter".
*   L'utilisateur appuie sur le bouton "Abandonner l’inscription".
