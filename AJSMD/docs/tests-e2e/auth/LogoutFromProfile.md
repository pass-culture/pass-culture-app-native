# Se déconnecter de l'application

Ce document décrit les étapes pour qu'un utilisateur se déconnecte de l'application.

## 1. Recherche d'un élément spécifique

*   L'application recherche un texte spécifique : "République Française - Liberté Égalité Fraternité".

## 2. Appui sur le bouton de déconnexion

*   L'utilisateur appuie sur le bouton "Déconnexion".

## 3. Vérification de l'écran de connexion

*   L'application vérifie que le bouton "Créer un compte" est affiché, ce qui confirme que l'utilisateur est déconnecté.
