# Vérifier le Solde du Profil

Ce document explique comment un utilisateur peut consulter son solde disponible dans l'application.

## 1. Accéder à la Page de Profil

*   L'utilisateur accède à l'application.
*   L'utilisateur appuie sur le bouton "Mon profil".

## 2. Récupération du Solde

*   L'application récupère et affiche le solde de l'utilisateur, un texte de la forme "XX,XX €".
*   L'application mémorise ce solde.

## 3. Retour à l'Accueil

*   L'utilisateur appuie sur le bouton "Accueil".

## 4. Vérification du Solde sur l'Accueil

*   L'écran d'accueil s'affiche, et l'application affiche un message du type "Tu as XX,XX € sur ton pass", affichant le solde précédemment récupéré.
