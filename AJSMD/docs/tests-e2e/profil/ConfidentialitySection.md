# Gérer les Préférences de Confidentialité

Ce document explique comment un utilisateur peut gérer ses préférences de confidentialité au sein de l'application.

## 1. Accéder aux Paramètres de Confidentialité

*   L'application vérifie si des commandes cloud doivent être exécutées.
*   L'utilisateur fait défiler l'écran.
*   L'utilisateur appuie sur le bouton "Confidentialité".
*   L'application affiche l'écran "Paramètres de confidentialité".

## 2. Accepter toutes les préférences

*   L'utilisateur appuie sur l'interrupteur "Tout accepter".
*   L'utilisateur fait défiler l'écran.
*   L'application affiche la "Politique de gestion des cookies".
*   L'utilisateur appuie sur le bouton "Enregistrer mes choix".
