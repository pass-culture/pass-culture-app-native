# Accéder à l'aide dans l'application

Ce document explique comment accéder à la section d'aide dans l'application.

## 1. Lancement et Recherche de l'Aide

*   L'utilisateur fait défiler l'écran.
*   L'application vérifie que le bouton "Aides" est visible.
*   L'utilisateur appuie sur le bouton "Nouvelle fenêtre : Centre d’aide".

## 2. Gestion des Premières Utilisations de Chrome (si nécessaire)

*   Si l'application est ouverte pour la première fois et affiche "Utiliser l'appli sans compte", l'utilisateur appuie sur ce bouton.
*   L'utilisateur appuie ensuite sur "Non, merci".
*   L'utilisateur appuie sur le bouton "Retour" pour revenir à l'écran précédent.

## 3. Retour à l'application principale (selon la plateforme)

*   **Pour les utilisateurs Android :** L'application effectue une action spécifique pour revenir à l'écran précédent.
*   **Pour les utilisateurs iOS :** L'utilisateur appuie sur le bouton "Revenir à pass Culture S".
