# Navigation vers les écrans de connexion et d'inscription

Ce document décrit les étapes pour naviguer vers les écrans de connexion et d'inscription de l'application.

## 1. Accès à l'écran de connexion

*   L'utilisateur appuie sur le bouton "Se connecter".
*   L'application vérifie que l'écran "Connecte-toi" s'affiche.

## 2. Retour en arrière après l'écran de connexion

*   **Pour Android :** L'utilisateur appuie sur le bouton "Retour" de son appareil.
*   **Pour iOS :** L'utilisateur appuie sur le bouton "Revenir en arrière".
*   **Pour le Web :** L'utilisateur appuie sur le bouton "Revenir en arrière".

## 3. Accès à l'écran d'inscription

*   L'utilisateur appuie sur le bouton "Créer un compte".
*   L'application vérifie que l'écran "Crée-toi un compte" s'affiche.

## 4. Retour en arrière après l'écran d'inscription

*   **Pour Android :** L'utilisateur appuie sur le bouton "Retour" de son appareil.
*   **Pour iOS :** L'utilisateur appuie sur le bouton "Revenir en arrière".
*   **Pour le Web :** L'utilisateur appuie sur le bouton "Revenir en arrière".

## 5. Fin de la navigation

*   L'application attend la fin de toutes les animations en cours.
