# Navigation sur la page d'accueil

Ce document décrit le parcours d'un utilisateur sur la page d'accueil de l'application.

## 1. Arrivée sur la page d'accueil

*   L'utilisateur arrive sur la page d'accueil de l'application.
*   L'écran affiche "Mon profil".
*   L'écran affiche "Créer un compte".
*   L'écran affiche "Se connecter".
*   L'écran affiche "Paramètres de l’application".
*   L'écran affiche "Aides".
