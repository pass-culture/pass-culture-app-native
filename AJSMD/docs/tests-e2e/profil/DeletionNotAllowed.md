# Gérer la suppression de compte

Ce document explique les étapes qu'un utilisateur suit pour essayer de supprimer son compte dans l'application.

## 1. Tentative de Suppression de Compte

*   L'application affiche un message indiquant que la suppression de compte n'est pas encore disponible.
*   L'utilisateur appuie sur le message d'erreur.
*   L'application affiche les options suivantes : "Retourner sur mon profil" et "Désactiver mes notifications".
*   L'utilisateur appuie sur le bouton "Revenir en arrière".
