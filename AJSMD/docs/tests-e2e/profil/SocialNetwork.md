# Navigation dans les applications web depuis l'application principale

Ce document décrit le parcours d'un utilisateur qui ouvre des applications web directement depuis l'application principale, comme Instagram, TikTok et Facebook.

## 1. Ouverture d'Instagram

*   L'utilisateur appuie sur l'icône "Instagram".
*   L'application attend que l'animation de transition se termine.
*   **Si c'est la première fois que l'utilisateur ouvre Chrome, l'application propose d'accepter les conditions d'utilisation.**
    *   L'utilisateur appuie sur "Accepter et continuer".

## 2. Vérification et Retour dans l'Application Principale

*   L'application vérifie que la barre d'adresse de Chrome est visible.
*   L'application utilise une fonction spécifique pour revenir en arrière dans la navigation web.
*   L'utilisateur appuie sur "X" pour fermer l'onglet Instagram.
*   L'application attend que l'animation de transition se termine.
*   L'application vérifie que la barre d'adresse de Chrome est visible.
*   L'application utilise une fonction spécifique pour revenir en arrière dans la navigation web.

## 3. Ouverture de TikTok

*   L'utilisateur appuie sur l'icône "TikTok".
*   L'application attend que l'animation de transition se termine.
*   L'application vérifie que la barre d'adresse de Chrome est visible.
*   L'utilisateur appuie deux fois sur le bouton "Retour" du téléphone pour quitter TikTok et revenir à l'application principale.

## 4. Gestion des Erreurs et Retour au Profil

*   **Si un message de bienvenue spécifique (avec "Bonjour Test-e2e") est visible, cela indique un problème et l'application effectue une action alternative.**
    *   L'utilisateur appuie sur "Mon profil".

## 5. Ouverture de Facebook

*   L'utilisateur appuie sur l'icône "Facebook".
*   L'application attend que l'animation de transition se termine.
*   L'utilisateur appuie sur le bouton "Retour" du téléphone pour quitter Facebook et revenir à l'application principale.

## 6. Accès aux Paramètres du Compte

*   L'utilisateur fait défiler l'écran jusqu'à ce qu'il voit l'option "Paramètres du compte".
