# Vérification des Préférences de Notifications

Ce document décrit les étapes pour vérifier les paramètres de notifications dans l'application.

## 1. Accéder aux Paramètres de Notifications

*   L'utilisateur accède aux paramètres de notifications.
    *   Sur iOS, l'utilisateur appuie sur le bouton "Notifications".
    *   Sur Android, l'utilisateur appuie sur le bouton "Notifications" (le second s'il existe plusieurs boutons avec le même nom).

## 2. Vérification des Paramètres Désactivés

*   L'application vérifie que les options suivantes sont désactivées :
    *   "Autoriser l’envoi d’e-mails"
    *   "Autoriser les notifications"
    *   "Suivre tous les thèmes"
    *   "Cinéma"
    *   "Lecture"
    *   "Musique"

## 3. Affichage des Autres Thèmes

*   L'utilisateur fait défiler l'écran pour voir d'autres options.
*   L'application vérifie que les options suivantes sont désactivées :
    *   "Spectacles"
    *   "Activités créatives"
    *   "Visites et sorties"

## 4. Retour en Arrière

*   L'utilisateur appuie sur "Revenir en arrière" pour quitter les paramètres de notifications.
