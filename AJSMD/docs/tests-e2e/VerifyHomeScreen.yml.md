# Découverte du Crédit Pass Culture

Ce document explique comment un nouvel utilisateur découvre son crédit Pass Culture et l'écran de bienvenue dans l'application.

## 1. Arrivée sur l'écran d'accueil

*   L'application affiche l'écran de bienvenue.
*   L'application affiche le texte "Bienvenue !"
*   L'application affiche le texte "Débloque ton crédit".
