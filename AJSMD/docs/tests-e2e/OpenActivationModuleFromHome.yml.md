# Déblocage de Crédit - Premier Accès

Ce document décrit les étapes qu'un utilisateur suit pour débloquer son crédit pour la première fois dans l'application.

## 1. Écran d'accueil

*   L'utilisateur arrive sur l'écran d'accueil et voit un message de bienvenue.
*   L'application affiche ensuite l'invitation à débloquer le crédit.
*   L'utilisateur attend la fin de l'animation de l'écran d'accueil.

## 2. Début du processus de déblocage

*   **Sur Android :** L'utilisateur appuie sur le bouton "Débloque ton crédit".
*   **Sur iOS :** L'utilisateur appuie sur le bouton "Débloque ton crédit" (représenté par un bouton bicolor).

## 3. Écran d'inscription

*   L'application affiche l'écran "Inscription".
*   L'application affiche le message "Crée-toi un compte".
