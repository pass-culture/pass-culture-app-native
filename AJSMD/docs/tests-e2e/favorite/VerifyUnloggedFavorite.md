# Se Connecter à l'Application

Ce document explique comment un utilisateur se connecte à l'application pass Culture.

## 1. Écran d'accueil

*   L'utilisateur arrive sur l'écran d'accueil.
*   L'écran affiche le texte "Identifie-toi pour retrouver tes favoris".
*   L'écran affiche les boutons "Créer un compte" et "Se connecter".
