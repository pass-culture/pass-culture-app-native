# Gérer ses Favoris dans l'Application

Ce document explique comment un utilisateur peut gérer ses éléments favoris dans l'application.

## 1. Vérification de l'Affichage des Favoris

*   L'application affiche l'onglet "Mes favoris".
*   L'application vérifie que l'onglet "Mes favoris" est visible.

## 2. Suppression des Favoris

*   L'application supprime tous les éléments des favoris.
    *   *Si un élément "Supprimer..." est visible, l'utilisateur appuie dessus pour le retirer des favoris. Cette action se répète jusqu'à ce qu'il n'y ait plus d'éléments à supprimer.*

## 3. Découverte du Catalogue

*   L'utilisateur appuie sur le bouton "Découvrir le catalogue".
