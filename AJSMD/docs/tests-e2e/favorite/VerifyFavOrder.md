# Trier les favoris

Ce document décrit comment un utilisateur peut trier sa liste de favoris dans l'application.

## 1. Accéder à l'option de tri

*   L'utilisateur arrive sur l'écran des favoris.
*   L'application affiche le bouton "Trier".
*   L'utilisateur appuie sur le bouton "Trier".

## 2. Choisir un critère de tri

*   L'écran affiche une liste d'options de tri : "Trier par Ajouté récemment", "Trier par Prix croissant", "Trier par Proximité géographique".
*   L'utilisateur sélectionne "Trier par Prix croissant".
*   L'utilisateur appuie sur le bouton "Valider".

## 3. Vérification du tri par prix

*   L'écran affiche "3 favoris" (ce qui indique que le tri a bien eu lieu).
*   L'utilisateur appuie à nouveau sur le bouton "Trier".

## 4. Choisir le tri par proximité géographique

*   L'utilisateur sélectionne "Trier par Proximité géographique".
*   L'utilisateur appuie sur le bouton "Valider".

## 5. Vérification du tri par proximité

*   L'écran affiche "3 favoris" (ce qui indique que le tri a bien eu lieu).
*   L'utilisateur appuie à nouveau sur le bouton "Trier".

## 6. Choisir le tri par ajout récent

*   L'utilisateur sélectionne "Trier par Ajouté récemment".
*   L'utilisateur appuie sur le bouton "Valider".
*   L'écran affiche "3 favoris" (ce qui indique que le tri a bien eu lieu).
