# Annuler une Réservation

Ce document décrit les étapes pour annuler une réservation effectuée dans l'application.

## 1. Démarrage et Recherche d'une Offre

*   L'utilisateur arrive sur l'écran principal de l'application et cherche une offre.
*   L'application affiche le bouton "Rechercher".
*   L'utilisateur appuie sur "Ma position".
*   L'utilisateur sélectionne "France entière".
*   L'application affiche les résultats de la recherche.

## 2. Sélection et Réservation de l'Offre

*   L'utilisateur sélectionne la première offre disponible.
*   L'utilisateur appuie sur le bouton "Réserver l’offre".
*   L'application affiche les "Détails de la réservation".
*   L'utilisateur accepte les conditions générales en cochant la case.
*   L'utilisateur fait défiler l'écran jusqu'à voir "Confirmer la réservation".
*   Sur Android, l'utilisateur doit faire défiler l'écran pour voir "Confirmer la réservation".
*   L'utilisateur appuie deux fois sur "Confirmer la réservation".
*   L'application affiche le message "Réservation confirmée\_!".
*   L'utilisateur appuie sur "Voir ma réservation".

## 3. Annulation de la Réservation

*   Sur iOS, si l'application affiche "Plus tard", l'utilisateur appuie sur "Plus tard".
*   L'utilisateur fait défiler l'écran pour voir "Annuler ma réservation".
*   L'utilisateur appuie deux fois sur "Annuler ma réservation".
