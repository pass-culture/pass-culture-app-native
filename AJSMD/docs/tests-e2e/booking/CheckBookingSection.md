# Gérer ses Réservations

Ce document décrit le parcours utilisateur pour consulter et annuler des réservations dans l'application.

## 1. Arrivée sur l'écran des réservations

*   L'utilisateur arrive sur l'écran principal de l'application.
*   L'application vérifie que les éléments suivants sont affichés : "Mes réservations", "En cours", et "Terminées".

## 2. Annulation d'une réservation "À retirer"

*   Si l'application affiche un bouton "À retirer...", l'utilisateur peut annuler sa réservation en cours.
    *   L'utilisateur appuie sur le bouton "À retirer...".
    *   L'utilisateur fait défiler l'écran.
    *   L'utilisateur fait à nouveau défiler l'écran.
    *   L'utilisateur appuie sur "Annuler ma réservation".
    *   L'utilisateur confirme en appuyant de nouveau sur "Annuler ma réservation".

## 3. Cas où aucune réservation n'est en cours

*   Si l'application affiche "Tu n’as pas de réservation en cours...", l'utilisateur voit un message lui indiquant l'absence de réservations en cours.
    *   L'application affiche le bouton "Découvrir le catalogue".
    *   L'utilisateur appuie sur "Terminées".
    *   L'application vérifie qu'un message de confirmation apparaît (par exemple "Réservation annulée...").
